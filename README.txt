# 🍔 Fat Diary — Calorie Counter

## Description
A simple calorie tracking app built with vanilla JavaScript (no frameworks, no build tools).
- Add foods with their calorie counts
- Auto-calculates daily total
- If you overeat ( > 2000 kcal ), the app starts panicking 😱
- Data is saved automatically in your browser's localStorage

## Tech Stack
- HTML
- CSS (responsive layout, animations, dark/light ready)
- JavaScript (DOM manipulation, localStorage, event handling)

## How to run
1. Download all files into one folder
2. Open `index.html` in your browser
3. Start adding foods and watch the magic happen 🍕

## What I learned / improved in this project
- Dynamic DOM generation with JavaScript (createElement, innerHTML)
- Working with localStorage to persist data
- Event handling (clicks, keyboard input, form validation)
- Conditional rendering (panic mode when calories exceed limits)
- Array methods (push, filter, forEach, reduce)

## Features
- Add/delete food entries
- Auto-save to localStorage
- Panic alerts when you eat too much
- Clean, responsive UI with hover effects and animations
- Keyboard support (Enter to add)
